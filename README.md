# Palm Archive Online Store

Palm Archive is a luxury fashion archive storefront. The website lets users browse designer products, search by brand or category, add items to a shopping bag, and message Palm Archive on Instagram to confirm size and availability.

## Main Features

* Clean fashion store homepage
* Brand pages for designer products
* Product search
* Shopping bag/cart
* Add and remove products from cart
* Instagram message link for ordering
* Product Finder chat assistant
* MongoDB product storage

## Tech Stack

This project uses HTML, CSS, JavaScript, Node.js, Express.js, MongoDB, and Mongoose.

## How to Run

Open the correct project folder that contains `package.json`.

Then run:

```bash
npm install
npm run dev
```

Open the website at:

http://127.0.0.1:4000

## Why I Built This Project

I built Palm Archive to practice full-stack web development while creating a realistic online store for my fashion archive brand idea. Since archive products often need size, condition, and availability confirmation, the website uses a shopping bag and Instagram message flow instead of a full checkout system.
